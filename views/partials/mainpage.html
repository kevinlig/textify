Textify requires a Storify URL. Using an embedded Storify feed? Find out how to get the Storify URL.

Embedded Storify Feeds

You can get the URL for an embedded Storify feed by clicking on the Storify logo in the top right corner of the Storify widget.

This will take you to the Storify web site. Copy this Storify page's URL into the Storify URL field in Textify.
